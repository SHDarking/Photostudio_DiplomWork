Програма содержит 4 строки подключения: 2 первые к локальным базам, 2 к облачным.
Облачные подключения содержат в название слово "Cloud".

Для проверки работ за исключением 5 практической обланых бд
будет достаточно и они уже немного заполнены тестовыми данными

Изменить строки подключения можно в MySQLDAO и MongoDAO для баз соответствующих названий.

Миграция, репликация, агрегация доступна исключительно из учетной записи администратора системы. 
При этом репликация и  агрегация исключительно для mongodb.
Изменить тип базы можно изменить в начале каждого контроллера.
